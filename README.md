# 📊 Blinkit Sales Dashboard - Power BI

![Power BI](https://img.shields.io/badge/Power%20BI-F2C811?style=for-the-badge&logo=powerbi&logoColor=black)

## 📌 Overview
Interactive Power BI dashboard analyzing **Blinkit grocery sales data** across multiple outlet types, sizes, and locations in India.

## 📈 Key Metrics
| Metric | Value |
|---|---|
| Total Sales | $1.20M |
| Average Sales | $140.99 |
| Avg Rating | 3.92 |
| No. of Items | 9K |

## 🔍 Features
- Sales trend by Outlet Establishment year (2012–2022)
- Item Type breakdown (Fruits, Snacks, Household, etc.)
- Outlet Size & Location tier analysis
- Fat Content segmentation (Low Fat vs Regular)
- Outlet Type comparison table

## 🛠️ Tools Used
- Power BI Desktop
- DAX Measures
- Dataset: BlinkIT Grocery Data
